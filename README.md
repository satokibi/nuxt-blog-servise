# my-first-nuxt-app

## Requirement
- Docker

## Usage

```shell
$ git clone https://github.com/satokibi/my-first-nuxt-app
$ cd my-first-nuxt-app
$ docker-compose up -d --build
$ docker-compose exec web /bin/ash
```

```terminal:コンテナ内
/app/vue-project # cd my-first-nuxt-app
/app/vue-project # npm run dev
```

http://localhost:3002
