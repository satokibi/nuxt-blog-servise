# nuxt-blog-service

## Requirement
- Docker

## Usage

```shell
$ git clone https://github.com/satokibi/nuxt-blog-service
$ cd nuxt-blog-service
$ docker-compose up -d --build
$ docker-compose exec web /bin/ash
```

```terminal:コンテナ内
/app/vue-project # cd nuxt-blog-service
/app/vue-project # npm run dev
```

http://localhost:3002
